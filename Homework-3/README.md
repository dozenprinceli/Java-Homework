# 第三次上机作业

>**项目名称：**

类的继承与接口

>**项目要求：**

1. 类的继承定义、属性隐藏和方法覆盖
2. 子类构造方法的定义，在子类构造方法中调用父类的构造方法
3. 理解在子类中调用父类的构造方法的原则与方法
4. 最终类、最终方法、抽象类、抽象方法的定义
5. Java基础类库中的一些常见类，如String使用
6. 接口的定义与实现、接口的扩展

**共7题，题目详见每题的README.md**
