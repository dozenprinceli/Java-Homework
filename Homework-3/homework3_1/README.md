### 1. 编写实现一个名为Person的类和它的子类Employee，Employee有两个子类Faculty和Staff。
> 具体要求
* Person类中的属性有：姓名name（String类型），地址address（String类型），电话号码telphone（String类型）和电子邮件地址email（String类型）；
* Employee类中的属性有：办公室office（String类型），工资wage（double类型），受雇日期hiredate（String类型）；
* Faculty类中的属性有：学位degree（String类型），级别level（String类型）；
* Staff类中的属性有：职务称号duty（String类型）。
